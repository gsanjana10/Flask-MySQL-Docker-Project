# 🚀 Flask + MySQL + Docker Compose Project

## 📌 Project Overview

This project demonstrates how to containerize a Flask web application using Docker and Docker Compose. It showcases DevOps concepts such as containerization, service orchestration, and application deployment.

## 🛠️ Technologies Used

- Python
- Flask
- Docker
- Docker Compose
- MySQL
- Git
- GitHub

## ✨ Features

- Containerized Flask Application
- Docker Compose Configuration
- Multi-Container Setup
- Port Mapping
- Container Networking
- Easy Deployment
- DevOps Best Practices

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/yourusername/Flask-MySQL-Docker-Project.git
```

### Navigate to Project Folder

```bash
cd Flask-MySQL-Docker-Project
```

### Build and Run Containers

```bash
docker compose up --build

```
